Метод Монте-Карло базується на законі великих чисел. Тож чим більше кидків total_throws ми задаємо в програму, тим ближчими будуть отримані результати до теоретичних значень. Для експерименту значення total_throws встановили 36, але було помічено, що відхилення від даних наведених в теоретичній таблиці (похибка) становило ~10-20%. При збільшенні значення total_throws, похибка зменшувалась, при значенні 100000 вона склала ~0.1%.
